You've written algorithms, loops, and conditionals. But what happens when code goes WRONG? That's when you become a debugger — the detective of the coding world!

Bug

Bug — A mistake in code that makes it do the wrong thing. Finding and fixing bugs is called debugging.
Fun fact: The first computer "bug" was an actual moth stuck inside a computer in 1947! Engineers literally taped it to their notebook. The name stuck.

Basic Version

Find the Bug — You're a robot inspector! Each program below has ONE bug. Find it before the robot does the wrong thing!

Three kinds of bugs to watch for:
  • Sequence bug — steps in the wrong order
  • Missing-step bug — a step is left out
  • Wrong-value bug — a number, condition, or word is wrong

Teacher / Parent Note

This activity introduces debugging — the most-practiced skill in real software development. Kids learn that finding bugs is just as important as writing code, and that there are common bug patterns (sequence, missing-step, wrong-value) that show up over and over. Discussion prompt: Ask your child to describe a bug they spotted in real life this week — even outside of code (a wrong recipe step, a confusing sign, a broken routine).
