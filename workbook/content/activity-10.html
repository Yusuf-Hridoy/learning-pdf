Algorithms work because the world has PATTERNS — repeating shapes, growing numbers, hidden rules. Today, you become a pattern detective: find the rule, predict what's next!

Pattern

Pattern — A rule that tells you what comes next in a sequence. Find the rule = predict the future!
Example: Your weekly schedule has a pattern — school, school, school, school, school, weekend! Once you know the rule, you can predict any day.

How It Works — Three Types of Patterns

There are three main types of patterns. Once you know them, you can spot patterns almost anywhere!

1. REPEATING
the same thing happens over and over
Example:
Rule: The loop is "square, circle." → Next: square
2. GROWING
each step gets bigger by a rule
Example: 2, 4, 6, 8, ___
Rule: The rule is "+2 each time." → Next: 10
3. RULE-BASED
every step follows a hidden instruction
Example: 1, 4, 9, 16, ___
Rule: Each number is its position squared. → Next: 25
Detective Trick: Try to find the rule in just 2-3 items before you guess what's next!

Teacher / Parent Note

Pattern recognition is the underlying skill behind almost every algorithm — from sorting (Activities 8-9) to debugging (Activity 7), and beyond that into search engines, recommendation systems, and machine learning. The 3 pattern types kids learn here map directly to real CS structures: repeating ↔ loops, growing ↔ recursion, rule-based ↔ functions. Discussion prompt: Ask your child to predict a pattern about your family's week — when does laundry get done? when is takeout night? — and see if they can predict the next "event" based on the pattern.
